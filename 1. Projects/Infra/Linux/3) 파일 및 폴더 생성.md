# mkdir: 폴더 생성
![[Pasted image 20250528151028.png]]

- 폴더 생성
	- `mkdir <foldername>`
	- 한번에 여러 폴더 생성
		- `mkdir <forlername> <forlername> <forlername> ...`
		- 계층 구조로 한번에 여러 폴더 생성
		  `mkdir <forlername>/<forlername>/.. -p`
