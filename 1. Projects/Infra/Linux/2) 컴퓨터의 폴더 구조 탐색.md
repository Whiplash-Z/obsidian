# Home Directory
![[Pasted image 20250527103308.png]]
- 사용자 관련 정보가 들어있다.
	- 모든 사용자는 각자의 이름으로 된 서브 디렉토리가 있다.
	- 각 사용자의 폴더에는 파일, 사용자 데이터 등 특정 유저에 대한 모든 정보
- `/home` = `~` = home directory
